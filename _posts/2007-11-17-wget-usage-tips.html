Examples:

If you need only to download something from an url:

[bash]wget http://cristian-radulescu.ro/image.gif[/bash]

With user (abc) and password (abc123):

[bash]wget http://cristian-radulescu.ro/document.pdf --user=abc --password=abc123[/bash]

Download from an url list (list.txt):

[bash]wget -i list.txt[/bash]

When the connection fails you can continue:

[bash]wget -c http://cristian-radulescu.ro/arhiva.zip[/bash]

The default number of retries is set to 20. Set the number of retries to infinity:

[bash]wget -t 0 -c http://cristian-radulescu.ro/archive.zip[/bash]

Download all files from a directory:

[bash]wget -p http://cristian-radulescu.ro/directory/[/bash]

Download files with specific extensions (.jpg, .gif, .png):

[bash]wget -nd -r -l1 -A.jpg -A.gif -A.png http://cristian-radulescu.ro/pictures/[/bash]

(-nd no directory, -r recursive download, -l1 level 1, no subdirectories).

Shell script example:

If the files are named useng a pattern (like wallpaper_1.jpg, wallpaper_2.jpg, ..., wallpaper_284.jpg, wallpaper_285.jpg) you can use a simple shell script:

[bash]#!/bin/bash
for ((i = 1; i <= 285; i ++))
do
wget -c "http://cristian-radulescu.ro/wallpapers/wallpaper_"$i".jpg"
done[/bash]

or type in a console:

[bash]for ((i = 1; i <= 285; i ++)); do wget -c "http://cristian-radulescu.ro/wallpapers/wallpaper_"$i".jpg"; done[/bash]

More details:

[bash]wget --help[/bash]

or

[bash]man wget[/bash]

in a console.
